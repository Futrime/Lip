# lip

## Usage

```shell
lip [options]
```

## Description

lip is a general package manager.

When a lip executable file exists under .lip/tools/lip/, it will be executed instead of the built-in one.

## Options

- `-h, --help`

  Show help.

- `-V, --version`

  Show version and exit.

- `-v, --verbose`

  Show verbose output.

- `-q, --quiet`

  Show only errors.

- `--no-color`

  Disable color output.
