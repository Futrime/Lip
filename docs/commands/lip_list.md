# lip list

## Usage

```shell
lip list [options]

alias: ls
```

## Description

List installed tooths.

Tooths are listed in a case-insensitive sorted order.

## Options

- `-h, --help`

  Show help.

- `--upgradable`

  List upgradable tooths.

- `--json`

  Output in JSON format. (cannot be hidden with `--quiet`)
